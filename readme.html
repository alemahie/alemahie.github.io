MNE_Vision¶

MNE Vision is a free and open source software based on MNE_Python. We propose here a graphical interface for MNE, able to perform some of the most common tasks in EEG analysis and processing. While also proposing other tools such as the connectivity, and classification with machine learning.

The main objective is to propose a free alternative to EEGLAB which requires a MATLAB license to fully work.

A video of presentation of the software is available here.

Documentation¶

MNE_Vision documentation is available here.

Installing and using MNE_Vision¶

There are two ways to install MNE Vision.

First, you can download it by doing in a terminal :

$ pip install mne_vision

This requires Python to be installed, if you don’t have python installed you can download it from here.

Once MNE Vision is downloaded, you can launch it from a terminal by doing :

$ mne_vision

The second way to download MNE Vision is through the GitHub page (here.).

You will first need to install his dependencies. To do so you can you can open a terminal in the directory where MNE_Vision is stored and type :

$ pip install requirements.txt

Then, MNE Vision can be launch in a terminal from the directory “pathtoMNE_VISION” and type :

$ python3 src/main.py

Note : Depending on how python is installed, python3 might need to be replaced by python if python3 is not reconised.

Dependencies¶

The required dependencies to run MNE-Python are:

  • Python >= 3.9.13
  • eeglabio
  • keras
  • matplotlib
  • mne >= 1.0.3
  • mne_connectivity >= 0.3
  • numpy
  • pymatreader
  • PyQt5
  • PyQt5-stubs
  • pyriemann
  • pyvista
  • pyvistaqt
  • scikit_learn
  • scipy
  • seaborn
  • tensorflow
